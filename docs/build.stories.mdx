## Kotlin Source

### Build
```
./gradlew clean build
```

### Publish Source

 * Declare sonatype credentials
```
export sonatypeUsername=
export sonatypePassword=
export signingPassword=
export signingKey=
```
 * Publish
```
VERSION=0.1.0-SNAPSHOT ./gradlew build publish
```
